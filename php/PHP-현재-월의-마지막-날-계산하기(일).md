# PHP 현재 월의 마지막 날 계산하기(일)

## 현재 월의 마지막 날 계산하기
```
  $today = date("t");
  echo $today;
```

28 ~ 31 이런 식으로 마지막 날이 나온다.


## 특정 월의 마지막 날 계산하기
```
  date("t", mktime(0, 0, 0, 9, 1, 2019)); //시간, 분, 초, 월, 일, 연도 (순서)
  echo $today;
```

혹은

```
  date("t", strtotime("2019-09-01"));
  echo $today;
```

이런 식으로 하면 해당 월의 말일을 구할 수 있다.
